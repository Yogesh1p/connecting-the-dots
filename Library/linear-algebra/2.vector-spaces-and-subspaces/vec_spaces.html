Vector Spaces

Vector Space

A set $V$ is a vector space if, for every $\vec{v}, \vec{w} \in V$ and every real scalar $c, d \in \mathbb{R}$, the linear combination stays inside the space:

$$ c\vec{v} + d\vec{w} \in V $$

This is called closure under addition and scalar multiplication. If you can add any two objects in the set and multiply them by numbers without leaving the set, you have a vector space.

The standard example is $\mathbb{R}^n$, the space of all real column vectors with $n$ components. However, linear algebra extends far beyond column vectors. Other valid vector spaces include:

  • $M_{m \times n}$: The space of all $m \times n$ matrices. (You can add matrices and multiply them by constants).
  • $P_n$: The space of all polynomials of degree $n$ or less.
  • $Z$: The space consisting of only the zero vector, $\{\vec{0}\}$. $c\vec{0} + d\vec{0} = \vec{0}$, which is still in the space.

Subspaces

A subspace is a vector space that sits inside a larger vector space. It must independently satisfy the rules of a vector space.

Subspace

A subset $W \subseteq V$ is a subspace if it satisfies three rules:

  1. Contains the zero vector: $\vec{0} \in W$ (Crucial! Set $c=0$).
  2. Closed under addition: $\vec{u}, \vec{v} \in W \Rightarrow \vec{u} + \vec{v} \in W$.
  3. Closed under scalar multiplication: $\vec{u} \in W, c \in \mathbb{R} \Rightarrow c\vec{u} \in W$.
The Geometry of Subspaces in $\mathbb{R}^3$

What do subspaces of $\mathbb{R}^3$ look like geometrically? There are exactly four types:

  • 3D: The whole space $\mathbb{R}^3$ itself.
  • 2D: Any plane that passes through the origin.
  • 1D: Any line that passes through the origin.
  • 0D: The origin itself $Z = \{(0,0,0)\}$.

Note: A plane or line that does not pass through the origin is NOT a subspace, because it fails to contain $\vec{0}$.

The Column Space and $Ax = b$

If we have a matrix $A$ of size $m \times n$, it acts as a mapping from $\mathbb{R}^n$ to $\mathbb{R}^m$. The most important subspace associated with $A$ is its Column Space, $C(A)$.

The Column Space $C(A)$

The column space of $A$ consists of all linear combinations of its columns. It lives inside the output space $\mathbb{R}^m$.

$$ C(A) = \{ Ax \mid x \in \mathbb{R}^n \} $$

The system $Ax = b$ asks: "Can I find a combination of the columns of $A$ (weights given by $x$) that produces the vector $b$?"

Existence of Solutions

The system $Ax = b$ has a solution if and only if $b$ is in the column space of $A$ ($b \in C(A)$).

The Null Space and Special Solutions

While the column space answers whether a solution exists, the Null Space answers how many solutions exist.

The Null Space $N(A)$

The null space of $A$ contains all solutions $x$ to the homogeneous equation $Ax = 0$. It lives in the input space $\mathbb{R}^n$.

Finding the Null Space: Pivot vs. Free Variables

To find the null space, we reduce $A$ to its Reduced Row Echelon Form, $R$. Elimination does not change the solutions to $Ax=0$, so $N(A) = N(R)$.

Suppose an $m \times n$ matrix $A$ has $r$ pivots.

  • There are $r$ pivot columns. Their corresponding variables are pivot variables.
  • There are $n - r$ columns without pivots. Their corresponding variables are free variables.

To find the Special Solutions that form the basis for $N(A)$: Set one free variable to $1$ and the rest to $0$. Solve for the pivot variables. Repeat this for each free variable. You will get exactly $n - r$ special solutions.

The Complete Solution to $Ax = b$

Linear algebra provides a beautiful structure for the complete solution to any system $Ax = b$. It is constructed in two parts.

Complete Solution Structure
$$ x_{complete} = x_p + x_n $$
  • $x_p$ is one particular solution to $Ax = b$. We usually find it by setting all free variables to $0$ and solving for the pivot variables.
  • $x_n$ represents all vectors in the null space $N(A)$. It is a linear combination of the Special Solutions.

Why does this work? If $Ax_p = b$ and $Ax_n = 0$, then $A(x_p + x_n) = Ax_p + Ax_n = b + 0 = b$.

Rank Condition Rows vs Cols Solutions to $Ax = b$ Null Space $N(A)$
Full Column Rank ($r=n$) $m \ge n$ (Tall) 0 or 1 (Unique) $Z = \{\vec{0}\}$
Full Row Rank ($r=m$) $m \le n$ (Wide) $\infty$ (Always exists) $n-m$ special solutions
Full Rank ($r=m=n$) $m = n$ (Square) Exactly 1 $Z = \{\vec{0}\}$
Rank Deficient ($r < m, r < n$) Any 0 or $\infty$ $n-r$ special solutions

Independence, Basis, and Dimension

Linear Independence

Vectors $v_1, v_2, \dots, v_n$ are linearly independent if the only combination that gives the zero vector is the trivial one:

$$ c_1v_1 + c_2v_2 + \dots + c_nv_n = 0 \implies c_1 = c_2 = \dots = c_n = 0 $$

Matrix definition: The columns of $A$ are independent if and only if $N(A) = \{\vec{0}\}$.

Basis and Dimension

A set of vectors is a Basis for a vector space $V$ if they possess two properties:

  1. They are linearly independent (not too many vectors).
  2. They span the space $V$ (not too few vectors).

The Dimension of a space is the number of vectors in every basis for that space.

Finding the basis of $C(A)$: Do not use the columns of $R$. Row operations change the column space! Instead, find the pivot columns of $R$, and select those exact same columns from the original matrix $A$.

Strang's Four Fundamental Subspaces

Every $m \times n$ matrix $A$ of rank $r$ creates four fundamental subspaces. The relationship between their dimensions is the most important theorem in linear algebra.

Subspace Symbol Lives in Dimension Basis comes from
Column space $C(A)$ $\mathbb{R}^m$ $r$ (rank) Pivot columns of $A$
Row space $C(A^T)$ $\mathbb{R}^n$ $r$ (rank) First $r$ rows of $R$ (or $U$)
Null space $N(A)$ $\mathbb{R}^n$ $n-r$ Special solutions to $Ax=0$
Left null space $N(A^T)$ $\mathbb{R}^m$ $m-r$ Special solutions to $A^Ty=0$
Why is it called the "Left" Null Space?

If $y$ is in $N(A^T)$, then $A^T y = 0$. If we take the transpose of both sides, we get:

$$ y^T A = 0^T $$

Here, $y^T$ is a row vector multiplying $A$ from the left. It represents the linear combinations of the rows of $A$ that result in a row of zeros.

The Fundamental Theorem of Linear Algebra (Part 1)

The column space and the row space always have the exact same dimension: $r$.

The dimension of the domain is $n$. The row space and null space split this domain:

$$ \dim(C(A^T)) + \dim(N(A)) = r + (n - r) = n $$

This equation is widely known as the Rank-Nullity Theorem.

Dimension of Matrix Spaces

Matrices themselves can be treated as vectors inside matrix spaces $M_{m\times n}$.

Dimension of $M$

Each $m \times n$ matrix has $mn$ independent entries, so the standard basis consists of $mn$ matrices (each with a single $1$ and the rest $0$'s).

$$ \dim(M_{m\times n}) = mn $$

Example: For $3 \times 3$ matrices, $\dim(M) = 9$.

We can find subspaces within square matrices $M_{n \times n}$:

  • Symmetric Matrices ($S$): $A^T = A$. The dimension is $\frac{n(n+1)}{2}$. For a $3 \times 3$, $\dim(S) = 6$.
  • Upper Triangular Matrices ($U$): The dimension is also $\frac{n(n+1)}{2}$.
  • Diagonal Matrices ($D$): These represent the intersection of $S$ and $U$ ($S \cap U$). The dimension is $n$.
