What Is a Positive Definite Matrix?

Some matrices behave like the number $+1$ — they always push things in a positive direction, never zero, never negative. Positive definite matrices are exactly this idea made precise for higher dimensions.

The central object is a quadratic form: for a square $n\times n$ matrix $A$ and a vector $x \in \mathbb{R}^n$, define

$$f(x) = x^T A x$$

This is a scalar — a single number that depends on both $A$ and the direction you choose for $x$. You can think of it geometrically: it measures how much the matrix stretches and rotates $x$ before projecting back onto $x$ itself. When this number is always positive (for every nonzero direction you could pick), the matrix is called positive definite.

Core Definition

A real $n \times n$ matrix $A$ is:

  • Positive definite (PD) if $x^TAx > 0$ for every nonzero $x \in \mathbb{R}^n$.
  • Positive semidefinite (PSD) if $x^TAx \geq 0$ for every $x \in \mathbb{R}^n$ — equality at zero is now allowed.
  • Indefinite if $x^TAx$ takes both positive and negative values for different choices of $x$.

Positivity of the quadratic form is the definition itself — every other test (eigenvalues, pivots, determinants) is a consequence that follows from it.

A Simple 2×2 Example

Consider the matrix $S = \begin{bmatrix} 2 & 1 \\ 1 & 2 \end{bmatrix}$. Its quadratic form is:

$$f(x, y) = x^T S x = 2x^2 + 2xy + 2y^2$$

Is this always positive? Complete the square: $2x^2 + 2xy + 2y^2 = 2\!\left(x + \tfrac{1}{2}y\right)^2 + \tfrac{3}{2}y^2$. Both terms are non-negative, and they cannot both be zero unless $x = y = 0$. So $S$ is positive definite.

Now swap the off-diagonal to $-3$: $S' = \begin{bmatrix} 2 & -3 \\ -3 & 2 \end{bmatrix}$. Try $x = (1, 1)$: $f(1,1) = 2 - 6 + 2 = -2 < 0$. That single test vector kills positive definiteness — $S'$ is indefinite.

Why Does It Matter?

Positive definite matrices show up everywhere that a notion of "upward-curving" or "stable" is needed:

  • Optimization: The Hessian at a local minimum is PD. If it isn't, you're at a saddle point, not a minimum.
  • Statistics: Covariance matrices are always PSD — they measure spread, which is never negative.
  • Least squares: The normal equations $A^T A x = A^T b$ have a unique solution exactly when $A^T A$ is PD.
  • Numerical methods: PD systems can be solved with the faster, more stable Cholesky factorization instead of full LU.

How Do You Actually Check? — Cases by Matrix Shape

The definition — $x^TAx > 0$ for all nonzero $x$ — is not something you can verify directly. You can't test every vector. So in practice, checking positive definiteness always comes down to one of a small number of equivalent conditions. Which one you use depends on the shape of the matrix in front of you. We work through all three cases.

Case 1 — $A$ is Rectangular ($m\times n$, $m\neq n$)

A rectangular matrix cannot be PD or PSD on its own, because $x^TAx$ is not even defined: $Ax$ lives in $\mathbb{R}^m$ while $x$ lives in $\mathbb{R}^n$, so the inner product $x^T(Ax)$ would mix spaces. Rectangular matrices enter the story only through the Gram matrix construction.

The Gram Matrix $A^TA$

Given any $m\times n$ matrix $A$ (with $m\geq n$), form the square $n\times n$ matrix $S=A^TA$. This is always symmetric and always PSD. It is PD if and only if $A$ has full column rank (i.e., the columns of $A$ are linearly independent, equivalently $\operatorname{null}(A)=\{0\}$).

Gram Matrix Test

Let $A$ be $m\times n$ with $m\geq n$.

$$x^T(A^TA)x = (Ax)^T(Ax) = \|Ax\|^2 \geq 0$$

Strict inequality $\|Ax\|^2>0$ for all $x\neq 0$ holds exactly when $Ax=0\Rightarrow x=0$, i.e., when $\operatorname{rank}(A)=n$.

So the practical test for a huge rectangular matrix $A$ is: compute the rank of $A$. If $\operatorname{rank}(A)=n$ (full column rank), then $A^TA$ is PD. Otherwise $A^TA$ is only PSD, with a null space of dimension $n-\operatorname{rank}(A)$.

How to find the rank of a large matrix: Run Gaussian elimination on $A$ and count the number of nonzero pivots. This takes $O(mn^2)$ work (assuming $m\geq n$). For truly huge matrices, a Singular Value Decomposition (SVD) gives the rank numerically — the number of singular values above a chosen tolerance. Rank deficiency shows up as one or more singular values equal (or close) to zero.

Why does this matter? In least-squares problems you solve the normal equations $A^TAx=A^Tb$. The solution is unique if and only if $A^TA$ is PD — which holds precisely when the columns of $A$ are independent. If $A^TA$ is only PSD, the system has infinitely many solutions and you need a pseudoinverse.

Case 2 — $A$ is Square but Not Symmetric

Now $A$ is $n\times n$ and the quadratic form $f(x)=x^TAx$ is well-defined. The first key insight is that the skew-symmetric part of $A$ contributes nothing to $f$.

Decomposing $A$ into Symmetric and Skew-Symmetric Parts

Any square matrix $A$ splits uniquely as:

$$A = S + K, \qquad S=\frac{A+A^T}{2},\quad K=\frac{A-A^T}{2}$$

where $S$ is symmetric ($S^T=S$) and $K$ is skew-symmetric ($K^T=-K$). For any vector $x$, the skew-symmetric part satisfies $x^TKx=0$ (because $x^TKx$ is a scalar equal to its own negative). Therefore:

$$x^TAx = x^TSx + x^TKx = x^TSx$$
Reduction to the Symmetric Part

A square (possibly non-symmetric) matrix $A$ is PD (resp. PSD) if and only if its symmetric part $S=\frac{A+A^T}{2}$ is PD (resp. PSD).

This means eigenvalues of $A$ alone cannot determine PD/PSD. A non-symmetric matrix can have all positive eigenvalues yet fail to be PD (its symmetric part may have a negative eigenvalue). Conversely, a matrix can be PD while having complex eigenvalues. The eigenvalues of $A$ are irrelevant; only the eigenvalues of $S=\frac{A+A^T}{2}$ matter.

Practical Test for a Large Square Non-Symmetric Matrix

For a huge $n\times n$ non-symmetric matrix $A$, proceed as follows:

  1. Form $S=\frac{A+A^T}{2}$. (This is $O(n^2)$ work — just average each entry with its mirror.)
  2. Apply the tests for symmetric matrices to $S$ (described in Case 3 below).

You have now reduced the non-symmetric problem to a symmetric one at negligible cost.

Case 3 — $A$ is Square and Symmetric

This is the richest case, and the one where we have the most tools. Let $S$ be an $n\times n$ real symmetric matrix. Because $S$ is symmetric, it admits three classical tests for positive definiteness — and they are all equivalent. Passing any one guarantees all the others.

The Three Equivalent Tests for PD

A real symmetric $n \times n$ matrix $S$ is positive definite if and only if any one (and therefore all) of the following hold:

  1. All pivots are positive: Every pivot $d_k > 0$ in the $LDL^T$ factorization of $S$.
  2. All eigenvalues are positive: $\lambda_i > 0$ for $i = 1, \ldots, n$.
  3. All upper-left determinants are positive (Sylvester's criterion): Every leading principal minor $\Delta_k = \det(S_{1:k,\,1:k}) > 0$ for $k = 1, \ldots, n$.

The pivots are the ratios of consecutive leading minors ($d_k = \Delta_k / \Delta_{k-1}$), linking tests 1 and 3. Eigenvalues and pivots always share the same signs by Sylvester's law of inertia, linking tests 1 and 2. Each test below makes this precise.

Test 1 — Completing the Square and the Pivot Test ($LDL^T$)

Rather than checking $x^TSx>0$ directly by plugging in every possible vector, we rewrite the quadratic form as an explicit sum of squares. This algebraic process hides a stunning matrix connection: completing the square is exactly Gaussian elimination in disguise.

Let's use a classic 2×2 example. Consider the matrix $S$ and its quadratic form $f(x,y)$:

$$S = \begin{bmatrix}2&6\\6&20\end{bmatrix} \implies f(x,y) = 2x^2 + 12xy + 20y^2$$

If we use high-school algebra to complete the square, we get:

$$f(x,y) = 2(x+3y)^2 + 2y^2$$

Now, let's run standard Gaussian elimination on $S$. To clear the $6$ in the bottom left, we multiply the first row by $3$ and subtract it from the second row. This gives us our upper triangular matrix $U$, our lower triangular multiplier matrix $L$, and a diagonal matrix of our pivots $D$:

$$S \xrightarrow{\text{elimination}} U = \begin{bmatrix}2&6\\0&2\end{bmatrix}, \quad L = \begin{bmatrix}1&0\\\mathbf{3}&1\end{bmatrix}, \quad D = \begin{bmatrix}\mathbf{2}&0\\0&\mathbf{2}\end{bmatrix}$$
The Factorization Reveal

Look closely at the completed square alongside our matrices:

$$f(x,y) = \mathbf{2}(x + \mathbf{3}y)^2 + \mathbf{2}y^2$$
  • The numbers outside the squares ($\mathbf{2}$ and $\mathbf{2}$) are exactly the pivots (the diagonal entries of $D$).
  • The number inside the square ($\mathbf{3}$) is exactly the multiplier from elimination (the entry in $L$).

Why does this happen? Gaussian elimination factors the symmetric matrix as $S = LDL^T$. If we plug this factorization directly into our quadratic form, the mechanics are exposed:

$$x^TSx = x^T(LDL^T)x = (L^Tx)^T D (L^Tx)$$

Let $z = L^Tx$. For our matrix, $L^T = \begin{bmatrix}1&3\\0&1\end{bmatrix}$, so $L^T \begin{bmatrix}x\\y\end{bmatrix} = \begin{bmatrix}x+3y\\y\end{bmatrix}$. These are exactly the terms inside our squares! The quadratic form is literally just $z^TDz$:

$$d_1 z_1^2 + d_2 z_2^2 = \mathbf{2}(x+3y)^2 + \mathbf{2}y^2$$

The Conclusion: A matrix is Positive Definite if and only if all of its pivots are strictly positive. The positive pivots guarantee that the "sum of squares" equation only has positive coefficients, meaning the bowl must open upwards. If a pivot is negative, you have a minus sign in your sum of squares, creating a saddle point.

Test 2 — Eigenvalue Test

The spectral theorem guarantees that a real symmetric matrix has an orthonormal eigenbasis: $S=Q\Lambda Q^T$ where $Q$ is orthogonal and $\Lambda=\operatorname{diag}(\lambda_1,\ldots,\lambda_n)$. Setting $y=Q^Tx$:

$$x^TSx = y^T\Lambda y = \sum_{i=1}^n \lambda_i y_i^2$$

This sum is strictly positive for every nonzero $y$ if and only if every eigenvalue $\lambda_i>0$. Note how similar this looks to the pivot test ($d_1 z_1^2 + d_2 z_2^2$), but they are fundamentally different transformations. Pivots use a non-orthogonal shear ($L^T$), while eigenvalues use a rigid rotation ($Q^T$). The pivots and eigenvalues are not equal in general, but Sylvester's law of inertia guarantees they always have the same signs.

Cost for large matrices: Computing all eigenvalues via full eigendecomposition costs $O(n^3)$. However, for large sparse matrices, iterative methods (e.g., the Lanczos algorithm) can find just the smallest eigenvalue $\lambda_{\min}$ much faster. $S$ is PD if and only if $\lambda_{\min}>0$.

Test 3 — Sylvester's Criterion (Leading Principal Minors)

The leading principal minor of order $k$ is the determinant of the top-left $k\times k$ submatrix of $S$. Sylvester's criterion states that $S$ is PD if and only if all $n$ leading principal minors are positive:

$$\Delta_1 = s_{11} > 0,\quad \Delta_2 = \det\begin{bmatrix}s_{11}&s_{12}\\s_{21}&s_{22}\end{bmatrix}>0,\quad\ldots,\quad\Delta_n=\det(S)>0$$

The connection to pivots is direct: the $k$-th pivot equals $\Delta_k/\Delta_{k-1}$ (with $\Delta_0=1$). So all pivots positive $\Leftrightarrow$ all leading minors positive — these two tests are the same thing in different clothing.

Cost: Computing a single determinant via LU costs $O(n^3)$, and you need $n$ of them, so naively this is $O(n^4)$. In practice you compute one $LDL^T$ factorization ($O(n^3)$) and read off all pivots at once; the minor test is then obtained for free. For a 2×2 or 3×3 matrix, directly checking minors is the fastest approach.

Why the Three Tests Are Equivalent

The chain of equivalences runs as follows:

Eigenvalues $\leftrightarrow$ Pivots. The spectral theorem gives $S = Q\Lambda Q^T$, and the $LDL^T$ factorization gives $S = LDL^T$. Both decompose the quadratic form into a sum of squares: $\sum \lambda_i y_i^2$ (via $y = Q^Tx$) and $\sum d_k z_k^2$ (via $z = L^Tx$). The change of variable between $y$ and $z$ is invertible, so the signs of the coefficients cannot change — this is Sylvester's law of inertia. All $\lambda_i > 0$ if and only if all $d_k > 0$.

Pivots $\leftrightarrow$ Upper-left determinants. The $k$-th pivot is the ratio $d_k = \Delta_k / \Delta_{k-1}$, where $\Delta_k$ is the determinant of the top-left $k \times k$ submatrix and $\Delta_0 = 1$. This means:

$$d_1 = \Delta_1, \quad d_2 = \frac{\Delta_2}{\Delta_1}, \quad d_3 = \frac{\Delta_3}{\Delta_2}, \quad \ldots$$

If all pivots $d_k > 0$, the products telescope: $\Delta_k = d_1 d_2 \cdots d_k > 0$ for every $k$. Conversely if all $\Delta_k > 0$, then each ratio $\Delta_k / \Delta_{k-1} > 0$. The two conditions are identical.

So the complete equivalence chain for a real symmetric matrix is:

$$\underbrace{x^TSx > 0 \; \forall x \neq 0}_{\text{definition}} \;\Longleftrightarrow\; \underbrace{\lambda_i > 0 \; \forall i}_{\text{eigenvalue test}} \;\Longleftrightarrow\; \underbrace{d_k > 0 \; \forall k}_{\text{pivot test}} \;\Longleftrightarrow\; \underbrace{\Delta_k > 0 \; \forall k}_{\text{upper-left det.\ test}}$$
Proof: why the number of positive pivots equals the number of positive eigenvalues

Both the pivot test and the eigenvalue test rewrite the same quadratic form $x^TSx$ as a sum of squares — just in different coordinate systems. The number of positive coefficients can't change when you switch between equivalent coordinate systems. That's the whole proof, unpacked carefully below.

Step A — Elimination gives $z = L^T x$

Go back to the 2×2 example from above. Completing the square gave:

$$x^TSx = 2\!\left(x + 3y\right)^2 + 2y^2$$

We defined $z_1 = x + 3y$ and $z_2 = y$. Write those substitutions as a matrix equation:

$$\begin{bmatrix}z_1\\z_2\end{bmatrix} = \begin{bmatrix}1 & 3\\0 & 1\end{bmatrix}\begin{bmatrix}x\\y\end{bmatrix}$$

That matrix is exactly $L^T$ from the $LDL^T$ factorization, where $L = \begin{bmatrix}1&0\\3&1\end{bmatrix}$. The elimination multiplier $3$ appears inside the square and simultaneously inside $L$. So the substitution is just:

$$\boxed{z = L^T x}$$

And the quadratic form becomes a clean sum of squares weighted by the pivots:

$$x^TSx = z^TDz = d_1 z_1^2 + d_2 z_2^2$$

In general, each elimination step introduces one substitution of the form $z_k = x_k + \sum_{j>k}(\text{multiplier}) \cdot x_j$. Stack all of them and you get $z = L^Tx$ exactly. The pivots $d_k$ are the coefficients in front of each square.

Step B — Eigenvectors give $y = Q^T x$

The spectral theorem decomposes $S = Q\Lambda Q^T$. Plug this into the quadratic form:

$$x^TSx = x^T(Q\Lambda Q^T)x = (Q^Tx)^T \Lambda\, (Q^Tx)$$

Define $y = Q^Tx$. This is just expressing $x$ in the eigenvector coordinate system — standard basis becomes the columns of $Q$. The quadratic form becomes:

$$\boxed{x^TSx = y^T\Lambda y = \lambda_1 y_1^2 + \lambda_2 y_2^2 + \cdots + \lambda_n y_n^2}$$

The eigenvalues $\lambda_i$ are the coefficients in front of each square.

The key comparison

$$x^TSx \;=\; \underbrace{d_1 z_1^2 + d_2 z_2^2 + \cdots + d_n z_n^2}_{\text{pivots as coefficients, shear coord.}} \;=\; \underbrace{\lambda_1 y_1^2 + \lambda_2 y_2^2 + \cdots + \lambda_n y_n^2}_{\text{eigenvalues as coefficients, rotated coord.}}$$

These are two different representations of the same function. The change of variable from $z$ to $y$ is the composition $y = Q^Tx = Q^T(L^T)^{-1}z$, which is an invertible linear map. Sylvester's law of inertia says: any two ways of writing a quadratic form as a sum of squares must use the same number of positive coefficients, the same number of negative coefficients, and the same number of zeros — regardless of which invertible change of variables you use.

So the number of positive $d_k$'s must equal the number of positive $\lambda_i$'s. The number of negative pivots must equal the number of negative eigenvalues. And the number of zero pivots must equal the number of zero eigenvalues.

In particular: all pivots positive $\Longleftrightarrow$ all eigenvalues positive. $\square$

Coordinate geometry intuition

Think of the two representations as describing the same bowl from two different camera angles:

  • Elimination ($z = L^Tx$) tilts the axes by shearing — the new axes are no longer perpendicular, but the bowl's shape is unchanged.
  • Eigenvectors ($y = Q^Tx$) rotates to perfectly axis-aligned coordinates — the bowl looks cleanest here, with no cross terms.

Whether a direction "opens upward" or "opens downward" is a physical fact about the bowl. It cannot depend on which coordinate system you use to describe it.

Positive Semidefinite (PSD): The Boundary Case

Replace every strict inequality above with $\geq$ to get the PSD conditions:

Tests for PSD (Symmetric Case)

A real symmetric $n\times n$ matrix $S$ is positive semidefinite if and only if:

  1. $x^TSx\geq 0$ for all $x$ (at least one nonzero $x$ achieves equality).
  2. All eigenvalues $\lambda_i\geq 0$ (at least one eigenvalue equals zero).
  3. All pivots $d_i\geq 0$ — but zero pivots are allowed (indicating a singular matrix).
  4. All principal minors (not just leading ones) $\geq 0$. Note: this requires checking all $2^n-1$ principal submatrices, not just the $n$ leading ones — Sylvester's criterion does not extend to PSD.

Why Sylvester's criterion fails for PSD: A matrix can have all leading principal minors $\geq 0$ yet fail to be PSD. The standard counterexample is $\begin{pmatrix}0&0\\0&-1\end{pmatrix}$: its leading minors are $0\geq 0$ and $0\geq 0$, but the matrix is not PSD. For PSD you must check all $2^n-1$ principal minors — computationally expensive for large $n$.

Practical test for PSD of a large matrix: Compute the smallest eigenvalue $\lambda_{\min}$ (e.g., via Lanczos). If $\lambda_{\min}\geq 0$, the matrix is PSD.

Connection to Calculus: The Hessian Matrix

Positive definiteness is exactly the right tool for identifying minima of multivariable functions. For a function $f(x_1,\ldots,x_n)$, the matrix of all second partial derivatives is called the Hessian:

$$H = \begin{pmatrix} f_{x_1x_1} & f_{x_1x_2} & \cdots \\ f_{x_2x_1} & f_{x_2x_2} & \cdots \\ \vdots & & \ddots \end{pmatrix}$$

Since $f_{x_ix_j}=f_{x_jx_i}$ (Clairaut's theorem), the Hessian is always symmetric. At a critical point where all first derivatives vanish, the local behavior of $f$ is governed entirely by the Hessian:

Second Derivative Test (Multivariable)

Let $x^*$ be a critical point of $f$ (i.e., $\nabla f(x^*)=0$) and let $H$ be the Hessian evaluated at $x^*$.

  • $H$ is positive definite $\Rightarrow$ $x^*$ is a strict local minimum.
  • $H$ is negative definite $\Rightarrow$ $x^*$ is a strict local maximum.
  • $H$ is indefinite (has both positive and negative eigenvalues) $\Rightarrow$ $x^*$ is a saddle point.
  • $H$ is PSD or NSD $\Rightarrow$ the test is inconclusive.

For the 2×2 case, this collapses to the familiar calculus rule: the critical point is a minimum when $f_{xx}>0$ and $f_{xx}f_{yy}-f_{xy}^2>0$ — which is exactly Sylvester's criterion applied to $H$. The quantity $f_{xx}f_{yy}-f_{xy}^2$ is the determinant of $H$, the second leading principal minor.

Why this matters for large problems. In optimization — training neural networks, fitting statistical models, solving physics problems — you repeatedly check whether the Hessian at the current iterate is PD. If it is, gradient descent with the right step size converges to a local minimum. If it is indefinite, you are at a saddle point and need to escape. The fastest check in practice is: attempt Cholesky on $H$. If it fails, you have a saddle.

Geometry: Graphs, Level Sets, and Conic Sections

To truly understand the geometry of a quadratic form $f(x) = x^TSx$, we must distinguish between two different geometric objects: the graph and the level set.

  • The Graph ($z = x^TSx$): If $x \in \mathbb{R}^2$, the graph lives in 3D space ($\mathbb{R}^3$). This is the "bowl" or "saddle" surface we plotted earlier.
  • The Level Set ($x^TSx = c$): This is a horizontal slice of the graph at a specific height $c$ (usually we choose $c=1$). This slice lives in the original 2D space ($\mathbb{R}^2$) and forms our conic sections: ellipses and hyperbolas.

1. The Ellipse (Positive Definite)

Let's look at the standard equation of an ellipse centered at the origin in 2D, aligned with the $x$ and $y$ axes:

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$$

Here, $a$ and $b$ are the lengths of the semi-major and semi-minor axes. Now, let's see how our quadratic form $x^TSx = 1$ maps to this.

By the spectral theorem, a symmetric matrix can be diagonalized as $S = Q\Lambda Q^T$. Plugging this into our quadratic form gives $x^T(Q\Lambda Q^T)x = 1$.

If we define a new coordinate vector $y = Q^Tx$ (which means $y^T = x^TQ$), the equation simplifies to a purely diagonal form $y^T\Lambda y = 1$. Let's write out exactly what this looks like with vectors and matrices in 2D:

$$ \begin{bmatrix} y_1 & y_2 \end{bmatrix} \begin{bmatrix} \lambda_1 & 0 \\ 0 & \lambda_2 \end{bmatrix} \begin{bmatrix} y_1 \\ y_2 \end{bmatrix} = 1 $$

Multiplying the matrices out gives us the simple sum of squares:

$$ \begin{bmatrix} \lambda_1 y_1 & \lambda_2 y_2 \end{bmatrix} \begin{bmatrix} y_1 \\ y_2 \end{bmatrix} = 1 \implies \lambda_1 y_1^2 + \lambda_2 y_2^2 = 1 $$

To force this into the standard ellipse fraction format, we move the eigenvalues to the denominator (by taking their reciprocals) and express those denominators as squares:

$$\frac{y_1^2}{(1/\sqrt{\lambda_1})^2} + \frac{y_2^2}{(1/\sqrt{\lambda_2})^2} = 1$$
The Anatomy of the Ellipsoid

By writing it in this exact shape, we can just read the lengths of the semi-axes straight off the denominator. When $S$ is Positive Definite, the equation $x^TSx = 1$ describes an ellipse (or ellipsoid) where:

  • The directions of the axes are the eigenvectors of $S$ (the columns of $Q$, handled by our substitution $y = Q^Tx$).
  • The lengths of the semi-axes are $a = 1/\sqrt{\lambda_1}$ and $b = 1/\sqrt{\lambda_2}$.

Notice the inverse relationship: a large eigenvalue means a steep bowl, which results in a short axis for the level set. The quadratic form grows very fast in that direction, so it hits the height $z=1$ very quickly.

2. The Circle (Equal Eigenvalues)

What happens if $\lambda_1 = \lambda_2 = \lambda$? (This means $S = \lambda I$, a multiple of the identity matrix). The equation becomes:

$$\lambda y_1^2 + \lambda y_2^2 = 1 \implies y_1^2 + y_2^2 = \frac{1}{\lambda}$$

This is the standard equation of a circle $x^2 + y^2 = r^2$ with radius $r = 1/\sqrt{\lambda}$. Geometrically, the "bowl" is perfectly symmetrical (isotropic). Every direction is an eigenvector, so the level set is a perfect circle.

3. The Hyperbola (Indefinite Matrices)

What if $S$ is indefinite, meaning it has mixed eigenvalue signs? Suppose $\lambda_1 > 0$ and $\lambda_2 < 0$. The diagonalized level set equation $x^TSx = 1$ becomes:

$$\lambda_1 y_1^2 - |\lambda_2| y_2^2 = 1 \implies \frac{y_1^2}{(1/\sqrt{\lambda_1})^2} - \frac{y_2^2}{(1/\sqrt{|\lambda_2|})^2} = 1$$

This is the standard equation for a hyperbola. Because the matrix is indefinite, the graph $z = x^TSx$ is a saddle. If you slice a saddle horizontally at $z=1$, the intersection is a hyperbola opening along the axis of the positive eigenvalue.

4. Where are the Parabolas? (The Paraboloid)

You might wonder: if we have ellipses and hyperbolas, can $x^TSx = 1$ ever be a 2D parabola?

The answer is no. A pure quadratic form $x^TSx$ is strictly centered at the origin (it contains only $x^2, y^2,$ and $xy$ terms). To get a 2D parabola, the general conic equation requires a linear term (like $y = ax^2$, which has a plain $y$).

To see a parabola in the domain, we have to look at the general quadratic function:

$$f(x) = x^TSx + b^Tx + c = 0$$

If $S$ is positive semidefinite with a zero eigenvalue (e.g., $\lambda_1 > 0, \lambda_2 = 0$), the quadratic form lacks a $y_2^2$ term. If the linear $b^Tx$ term provides a non-zero slope in the $y_2$ direction, the resulting level curve is a parabola.

However, the graph of a positive definite form ($z = x^TSx$) in 3D is a paraboloid. If you slice this 3D bowl vertically (e.g., set $y=0$), you get $z = \lambda_1 x^2$, which is exactly a 2D parabola.

Decision Guide: Which Test to Use and When

For a huge matrix in practice, the choice of test depends on structure:

  • Rectangular $A$ ($m\times n$): Check whether $\operatorname{rank}(A)=n$ using elimination or SVD. If yes, $A^TA$ is PD; if no, $A^TA$ is only PSD.
  • Dense square non-symmetric $A$: Form $S=\frac{A+A^T}{2}$ ($O(n^2)$), then apply any symmetric test to $S$.
  • Dense square symmetric $S$, small–medium $n$: $LDL^T$ factorization is simplest. Check all pivots — positive means PD, any zero means PSD, any negative means indefinite.
  • Sparse square symmetric $S$, large $n$: Use a sparse $LDL^T$ solver (e.g., CHOLMOD). Attempt Cholesky factorization $S=R^TR$; if it succeeds (no negative square roots encountered), $S$ is PD. This is the standard industrial test.
  • Need eigenvalue information anyway: Run the Lanczos algorithm to get $\lambda_{\min}$ and $\lambda_{\max}$ cheaply. Full eigendecomposition only if you need all $n$ eigenvalues.

Cholesky Factorization: The Industrial Test

The Cholesky factorization $S=R^TR$ (where $R$ is upper triangular with positive diagonal) exists if and only if $S$ is symmetric and positive definite. This makes it the definitive computational test: attempt the factorization, and if it completes without encountering a non-positive pivot, $S$ is PD.

$$S = R^TR, \qquad r_{kk}=\sqrt{d_k}>0$$

The Cholesky factor $R$ is related to the $LDL^T$ factorization by $R=D^{1/2}L^T$. It exists and has real entries if and only if all pivots $d_k>0$.

For a sparse PD matrix, Cholesky is about twice as fast as $LU$ decomposition and numerically extremely stable. Most scientific computing libraries (LAPACK's dpotrf, MATLAB's chol, NumPy's linalg.cholesky) implement this — a runtime exception or warning signals that $S$ is not PD.

Closure and Factorization Properties

Closure Under Addition

If $S$ and $T$ are symmetric PD $n\times n$ matrices, then $S+T$ is also symmetric PD. The proof is immediate: $(S+T)^T=S+T$ (symmetric), and $x^T(S+T)x=x^TSx+x^TTx>0+0=0$ for all nonzero $x$.

Three Standard Factorizations of a Symmetric PD Matrix

A symmetric PD matrix $S$ can always be written in three standard forms, each with its own use:

  • Rectangular factor: $S=A^TA$ for some $A$ with independent columns. Common in least-squares and statistics.
  • Cholesky factor: $S=R^TR$ with $R$ upper triangular and positive diagonal. The go-to for numerical linear algebra — fast, stable, and the standard PD test.
  • Symmetric square root: $S=A_3^2=A_3^TA_3$ where $A_3=Q\Lambda^{1/2}Q^T$ is itself symmetric PD. Used in statistics (whitening transforms), optimization (preconditioning), and quantum mechanics.
