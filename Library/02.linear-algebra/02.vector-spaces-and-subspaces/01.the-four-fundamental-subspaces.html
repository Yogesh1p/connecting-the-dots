The Four Fundamental Subspaces of a Matrix

Every $m \times n$ matrix $A$ is more than just a grid of numbers; it is a mathematical operation that transforms vectors from one space to another. Specifically, $A$ maps vectors in $\mathbb{R}^n$ (the domain) into $\mathbb{R}^m$ (the codomain). To truly understand what a matrix does, we must understand the four fundamental subspaces it creates.

1. Meet the Subspaces

Two of these subspaces live in the domain $\mathbb{R}^n$, and two live in the codomain $\mathbb{R}^m$. Let's define them mathematically.

The Four Spaces
Subspace Symbol Host Space Definition
Row Space $C(A^T)$ $\mathbb{R}^n$ All linear combinations of the rows of $A$.
Null Space $N(A)$ $\mathbb{R}^n$ All solutions $x$ to the equation $Ax = 0$.
Column Space $C(A)$ $\mathbb{R}^m$ All linear combinations of the columns of $A$.
Left Null Space $N(A^T)$ $\mathbb{R}^m$ All solutions $y$ to the equation $A^Ty = 0$.

2. Dimensions and the Fundamental Theorem (Part 1)

The true magic of linear algebra lies in the dimensions of these spaces. The number of independent rows in a matrix is always exactly equal to the number of independent columns. This number is the rank ($r$) of the matrix.

Because the rank is $r$, the Row Space and Column Space both have dimension $r$. The remaining dimensions in each host space are filled by the null spaces.

Fundamental Theorem of Linear Algebra, Part 1
  • $\dim C(A^T) = r$ (Dimension of Row Space)
  • $\dim N(A) = n - r$ (Dimension of Null Space)
  • $\dim C(A) = r$ (Dimension of Column Space)
  • $\dim N(A^T) = m - r$ (Dimension of Left Null Space)

Notice how nicely they pair up: in $\mathbb{R}^n$, the row space ($r$) and the null space ($n-r$) add up to the total dimension $n$. In $\mathbb{R}^m$, the column space ($r$) and the left null space ($m-r$) add up to the total dimension $m$.

3. Orthogonality (Part 2)

Not only do these pairs of subspaces neatly divide up the dimensions of $\mathbb{R}^n$ and $\mathbb{R}^m$, but they do so at right angles. They are orthogonal complements.

If $x$ is a vector in the null space, then $Ax = 0$. This literally means that the dot product of every row of $A$ with the vector $x$ is exactly zero. Therefore, every vector in the null space is perpendicular to every vector in the row space.

4. The Big Picture:

On the left side, we have the domain $\mathbb{R}^n$, split cleanly into the Row Space and the Null Space. On the right side, we have the codomain $\mathbb{R}^m$, split cleanly into the Column Space and the Left Null Space.

The True Action of a Matrix

The diagram visualizes exactly what happens when you multiply a vector $x$ by a matrix $A$:

  1. Every vector $x$ in $\mathbb{R}^n$ can be split into a row space piece ($x_r$) and a null space piece ($x_n$). So, $x = x_r + x_n$.
  2. When multiplied by $A$, the null space piece is utterly destroyed: $Ax_n = 0$.
  3. The row space piece is mapped directly into the column space. This mapping is invertible (one-to-one) from the Row Space to the Column Space.
  4. Therefore, $Ax = A(x_r + x_n) = Ax_r + 0 = b$. The result $b$ always lives entirely in the column space.
