When a vector $b$ is projected onto a line, its projection $p$ is the part of $b$ that lies perfectly along that line. If $b$ is projected onto a plane, $p$ is the part in that plane. To calculate these projections, we use a projection matrix $P$, such that $p = Pb$.

Consider $b = (2, 3, 4)$. If we project it onto the $z$-axis, we get $p_1 = (0, 0, 4)$. If we drop it straight down onto the $xy$-plane, we get $p_2 = (2, 3, 0)$. The projection matrix $A$ picks out the $z$-component, while $P_2$ picks out the $x$ and $y$ components:

$$ A = \begin{bmatrix} 0 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 1 \end{bmatrix}, \quad P_2 = \begin{bmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 0 \end{bmatrix} $$

In this scenario, $p_1$ and $p_2$ are perpendicular, and their respective subspaces ($z$-axis and $xy$-plane) are orthogonal complements. The sum of the projections perfectly reconstructs the original vector: $p_1 + p_2 = b$. Equivalently, the projection matrices sum to the identity matrix: $A + P_2 = I$.

1. Projection Onto a Line

Let a line go through the origin in the direction of a vector $a$. We want to find the point $p$ along that line that is closest to a given vector $b$. The key to finding this closest point is orthogonality: the error vector linking $p$ to $b$ must be perpendicular to the line.

We define the error vector as $e = b - p$. Because $p$ must lie on the line defined by $a$, it is some scalar multiple of $a$. We will call this scalar $\hat{x}$, so $p = \hat{x}a$.

Since the error $e$ is perpendicular to $a$, their dot product is zero:

$$ a^T (b - \hat{x}a) = 0 \implies a^T b - \hat{x} a^T a = 0 $$
The Projection Formula (1D)

Solving for the scalar $\hat{x}$ gives:

$$ \hat{x} = \frac{a^T b}{a^T a} $$

The projection of $b$ onto the line through $a$ is the vector $p$:

$$ p = \hat{x}a = a \frac{a^T b}{a^T a} $$

To find the projection matrix $P$, we rewrite the formula to isolate the matrix acting on $b$. By moving the scalar fraction to the right of $a$, we get $p = \frac{aa^T}{a^Ta} b$.

1D Projection Matrix
$$ P = \frac{aa^T}{a^T a} $$

Notice that the numerator $aa^T$ is a column times a row (which creates a full matrix), while the denominator $a^Ta$ is a row times a column (which evaluates to a scalar number).

If you double the vector $a$, the matrix $P$ stays exactly the same, because projecting onto a line doesn't change if you stretch the basis vector defining that line. Furthermore, if you square the matrix, $P^2 = P$. Projecting a second time doesn't change anything; the vector is already on the line.

2. Projection Onto a Subspace

Now we move beyond projecting onto a single line. Suppose we want to project onto an $n$-dimensional subspace in $\mathbb{R}^m$, defined by linearly independent basis vectors $a_1, \dots, a_n$. We place these basis vectors into the columns of a matrix $A$.

The combinations in $\mathbb{R}^m$ are the vectors $Ax$ in the column space. We are looking for the specific combination $p = A\hat{x}$ that is closest to a given vector $b$.

Just like the 1D case, the error vector $e = b - A\hat{x}$ must be perpendicular to the subspace. This means $b - A\hat{x}$ must make a right angle with every basis vector $a_1, \dots, a_n$:

$$ \begin{matrix} a_1^T(b - A\hat{x}) = 0 \\ \vdots \\ a_n^T(b - A\hat{x}) = 0 \end{matrix} $$

We can compactly write these $n$ equations by stacking the vectors $a_i^T$ as the rows of $A^T$. This yields the most important equation in data science and least squares:

The Normal Equation
$$ A^T (b - A\hat{x}) = 0 \implies A^T A \hat{x} = A^T b $$

The matrix $A^T A$ is an $n \times n$ symmetric matrix. As long as the columns of $A$ are linearly independent, $A^T A$ is invertible.

This normal equation beautifully mirrors the Fundamental Theorem of Linear Algebra. The error vector $b - A\hat{x}$ is perpendicular to the column space $C(A)$, which means it must live exactly in the left null space $N(A^T)$. Therefore, multiplying it by $A^T$ must yield zero.

3. The Subspace Projection Matrix

With the normal equation, we can compute everything in three logical steps:

  1. Find the coefficients $\hat{x}$:
    $\hat{x} = (A^T A)^{-1} A^T b$
  2. Find the projection vector $p$:
    $p = A\hat{x} = A(A^T A)^{-1} A^T b$
  3. Find the projection matrix $P$:
    $P = A(A^T A)^{-1} A^T$

Compare this $m \times m$ matrix to the 1D formula $P = \frac{aa^T}{a^Ta}$. The number $a^Ta$ becomes the matrix $A^TA$. When it is a number, we divide by it. When it is a matrix, we invert it.

Warning: Matrix Inversion

Do not attempt to simplify $(A^T A)^{-1}$ into $A^{-1} (A^T)^{-1}$. The matrix $A$ is rectangular ($m \times n$) and does not have an inverse. The inverse is only valid for the square product $A^T A$.

4. Why is $A^T A$ Invertible?

We rely heavily on $(A^T A)^{-1}$. This inversion is mathematically legal if and only if $A$ has linearly independent columns. Here is a quick proof of why:

Suppose $A^T A x = 0$. We want to prove this implies $x = 0$ (meaning the null space only contains the zero vector).

$$ x^T A^T A x = x^T 0 = 0 $$

Notice that $x^T A^T$ is the transpose of $Ax$. Thus, we can rewrite the equation as:

$$ (Ax)^T (Ax) = 0 \implies \|Ax\|^2 = 0 $$

If the length of a vector is zero, the vector itself must be zero. Therefore, $Ax = 0$. Since we assumed $A$ has independent columns, the only solution to $Ax = 0$ is $x = 0$. Because $x = 0$ is the only vector in the null space of $A^TA$, the matrix $A^TA$ is invertible.
