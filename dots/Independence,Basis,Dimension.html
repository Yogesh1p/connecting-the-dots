Every subspace has a true size — a single number that measures it completely. A matrix might have ten columns, but the column space they produce might only be three-dimensional. Finding that number is what this section is about.

The path runs through three ideas in sequence. Independence removes redundancy from a set of vectors. Spanning ensures the set reaches every corner of the space. A basis does both at once — it is the minimal complete description of a space. Count the vectors in any basis and you get the dimension. That is the true size.

1. Linear Independence

The core question is simple: does any vector in a set add something new, or can it be built from the others? If a vector is just a combination of the rest, it is redundant. A set with no redundant vectors is called linearly independent.

Definition: Linear Independence

Vectors $v_1, v_2, \dots, v_n$ are linearly independent if the only way to combine them into the zero vector is the trivial way — all coefficients zero:

$$x_1 v_1 + x_2 v_2 + \dots + x_n v_n = 0 \implies x_1 = x_2 = \dots = x_n = 0.$$

When the vectors are columns of a matrix $A$, this is equivalent to saying the only solution to $Ax = 0$ is $x = 0$. The null space $N(A)$ contains only the zero vector.

If any non-zero coefficients produce the zero vector, the vectors are linearly dependent.

Two concrete cases make the definition clear. The vectors $(1, 1)$ and $(-1, -1)$ are dependent — they lie on the same line through the origin, and the combination $1 \cdot (1,1) + 1 \cdot (-1,-1) = (0,0)$ uses non-zero coefficients. By contrast, $(1, 0)$ and $(0, 1)$ point in genuinely different directions: no non-zero combination of them produces zero, so they are independent.

A set containing the zero vector is always dependent, immediately. You can assign any non-zero coefficient to the zero vector and the sum is still zero — so the trivial combination is not the only one that works.

When dependence is guaranteed

If $A$ is $m \times n$ with $n > m$ — more columns than rows — then elimination produces at most $m$ pivots. The remaining $n - m$ columns are free. A free variable can be set to a non-zero value and solved for, giving a non-zero vector in the null space. So the columns are dependent. No exceptions.

Theorem

Any $n$ vectors in $\mathbb{R}^m$ are linearly dependent if $n > m$. When there are more vectors than the dimension of the space they live in, dependence is unavoidable.

This is why three vectors in $\mathbb{R}^2$ are always dependent, regardless of which three vectors you pick. And it is the exact reason why the columns of a matrix are independent only when rank $r = n$ — every column is a pivot column, there are no free variables, and the null space is $\{0\}$.

Testing independence: a worked example

Are the columns of this matrix independent?

$$A = \begin{bmatrix} 1 & 3 & 3 \\ 2 & 6 & 9 \\ -1 & -3 & 3 \end{bmatrix}$$

Run elimination. Row 2 minus $2 \times$ Row 1, then Row 3 plus Row 1:

$$A \to \begin{bmatrix} 1 & 3 & 3 \\ 0 & 0 & 3 \\ 0 & 0 & 6 \end{bmatrix} \to \begin{bmatrix} 1 & 3 & 3 \\ 0 & 0 & 3 \\ 0 & 0 & 0 \end{bmatrix}$$

Two pivots, not three. Column 2 is free. Setting $x_2 = 1$ and solving gives the special solution $x = (-3, 1, 0)$, which satisfies $Ax = 0$ with non-zero entries. The columns are dependent. The special solution reveals exactly how: column 2 equals $3 \times$ column 1.

Two concrete cases make the definition clear. The vectors $(1, 1)$ and $(-1, -1)$ are dependent — they lie on the same line through the origin, and the combination $1 \cdot (1,1) + 1 \cdot (-1,-1) = (0,0)$ uses non-zero coefficients. By contrast, $(1, 0)$ and $(0, 1)$ point in genuinely different directions: no non-zero combination of them produces zero, so they are independent.

Think of linear independence as the requirement to explore something new. If you start with one random vector, you can move back and forth along it—you have a 1D space to explore. To take a step away from that line and discover something new, a second vector doesn't have to be perfectly perpendicular to the first. It just needs to have a component that is orthogonal to the line you are limited to. That orthogonal component is what pulls you out of 1D and grants you access to a 2D plane.

To expand into 3D space, you need a third vector with a component orthogonal to that entire 2D plane. Any vector without that orthogonal component is redundant—it leaves you trapped in the space you can already span, making it linearly dependent.

Initializing space...

A set containing the zero vector is always dependent, immediately. You can assign any non-zero coefficient to the zero vector and the sum is still zero...

2. Spanning a Space

Independence removes redundancy, but it says nothing about reach. A set of two independent vectors in $\mathbb{R}^3$ covers a plane — it cannot reach every point in three-dimensional space. To describe a whole space, vectors must span it.

Definition: Spanning

Vectors $v_1, \dots, v_k$ span a space $S$ if every vector in $S$ can be written as a linear combination of those vectors. The columns of a matrix always span its column space.

A spanning set does not need to be efficient. Three vectors can span $\mathbb{R}^2$ — one of them is just unnecessary. In fact, any spanning set that contains a dependent vector can have that vector removed and still span the same space. Spanning is about reach, not minimality.

The rows of $A$ span a subspace too: the row space $C(A^T)$, a subspace of $\mathbb{R}^n$. The columns span the column space in $\mathbb{R}^m$. Same numbers, different vectors, different spaces.

So we have two separate demands on a set of vectors: no redundancy (independence) and full reach (spanning). A basis is the set that satisfies both at once.

3. Basis

Definition: Basis

A sequence of vectors $v_1, v_2, \dots, v_d$ is a basis for a vector space if:

  1. The vectors are linearly independent — no redundancy.
  2. The vectors span the space — full reach.

Every vector in the space is a unique combination of the basis vectors.

The uniqueness follows directly from independence, and this algebraic proof translates to a beautiful geometric intuition. Imagine you want to reach a specific destination vector, $v$. If you have a true basis, those independent vectors provide a perfect coordinate system, guaranteeing there is exactly one unique path to get there.

What if there were two different ways to reach $v$? Suppose Path A uses scalars $a_1, \dots, a_d$ and Path B uses a different set of scalars $b_1, \dots, b_d$ to reach the exact same destination.

$$v = a_1 v_1 + \dots + a_d v_d \quad \text{(Path A)}$$ $$v = b_1 v_1 + \dots + b_d v_d \quad \text{(Path B)}$$

If you subtract Path B from Path A, geometrically this means you walk Path A forward to the destination, and then follow Path B in reverse back to the start. Because both paths connect the origin to the exact same spot, you create a closed loop and end up right back at the zero vector:

$$0 = (a_1 - b_1)v_1 + \dots + (a_d - b_d)v_d$$

This brings us directly back to the definition of linear dependence! If the paths were genuinely different, the coefficients would be different (e.g., $a_1 - b_1 \neq 0$). You would have a non-trivial combination equaling zero — a redundant "round trip." But if the vectors are truly independent, the only way a combination can equal zero is if every single coefficient is zero. This forces $a_i - b_i = 0$, meaning $a_i = b_i$. Path A and Path B were the exact same path all along.

The standard basis and beyond

The most natural basis for $\mathbb{R}^3$ is the columns of the identity matrix:

$$e_1 = \begin{bmatrix}1\\0\\0\end{bmatrix}, \quad e_2 = \begin{bmatrix}0\\1\\0\end{bmatrix}, \quad e_3 = \begin{bmatrix}0\\0\\1\end{bmatrix}$$

These are independent ($Ix = 0$ only when $x = 0$) and they span $\mathbb{R}^3$. But they are far from the only valid basis. The vectors $(1,1,0)$, $(0,1,1)$, and $(1,0,1)$ also form a basis for $\mathbb{R}^3$ — just a less obvious one. There are infinitely many choices.

The Invertibility Test

$n$ vectors in $\mathbb{R}^n$ form a basis if and only if they are the columns of an invertible $n \times n$ matrix. Invertibility, independence, and spanning $\mathbb{R}^n$ are all equivalent for square matrices.

Choosing a basis from a matrix: $A$ not $R$

When a matrix has dependent columns, elimination identifies which ones to keep. But there is a critical trap: the pivot columns of the echelon form $R$ are not the same vectors as the pivot columns of $A$. Row operations change the column space. The column space of $A$ and the column space of $R$ are generally different subspaces, even though they have the same dimension.

To see why: the columns of $R$ live in a rotated, scaled coordinate system produced by elimination. In the $QR$ factorization $A = QR$, the matrix $Q$ is an orthogonal matrix that rotates the column directions. So $C(A)$ and $C(R)$ have the same shape — same dimension, same number of independent directions — but they are tilted differently in space.

Which columns to use as a basis

Use elimination on $A$ to find which column numbers are pivot columns. Then take those columns from the original matrix $A$ — not from $R$. Those are the correct basis vectors for $C(A)$.

The row space is different: row operations preserve the row space, so the nonzero rows of $R$ directly give a valid basis for the row space of $A$.

4. Dimension

Bases are not unique — there are infinitely many for any space. Yet something stays fixed no matter which basis you pick: the number of vectors. This is the theorem that makes dimension well-defined.

Start with a concrete case. Two bases for $\mathbb{R}^2$: the standard basis $\{(1,0),\, (0,1)\}$ and the rotated basis $\{(1,1),\, (1,-1)\}$. Both have exactly 2 vectors. Could a third basis have 3 vectors? No — three vectors in $\mathbb{R}^2$ are always dependent by the theorem from Section 1, so they cannot form a basis. Could a basis have just 1 vector? No — one vector spans only a line, not the whole plane. The count is locked at 2. Always.

Theorem: All Bases Have the Same Size

If $v_1, \dots, v_m$ and $w_1, \dots, w_n$ are both bases for the same vector space, then $m = n$.

Proof sketch. Suppose $n > m$. Since the $v$'s span the space, each $w_j$ is a combination of the $v$'s. Arrange these combinations as columns of a matrix: it has one row per $v$ and one column per $w$, making it $m \times n$ with $n > m$ — short and wide. By the theorem from Section 1, $Ax = 0$ has a non-zero solution. That solution gives a non-zero combination of the $w$'s equal to zero, contradicting their independence. So $n > m$ is impossible. By symmetry, $m > n$ is also impossible. Therefore $m = n$.

Definition: Dimension

The dimension of a vector space is the number of vectors in every basis for that space.

A word on language: we say the rank of a matrix and the dimension of a subspace. "The dimension of a matrix" and "the rank of a space" have no meaning. What connects them is this: the dimension of the column space equals the rank of the matrix.

One practical consequence: if you know a space has dimension $d$ and you find $d$ independent vectors inside it, you are done. They automatically span the space and form a basis — no further checking needed.

5. The Fundamental Balance

For any $m \times n$ matrix $A$ with rank $r$, the column space and null space have dimensions that add up perfectly.

Column space $C(A)$

The pivot columns of $A$ are independent and span $C(A)$. There are $r$ of them:

$$\dim C(A) = r$$

Null space $N(A)$

Each free variable produces one special solution to $Ax = 0$. Each special solution has a $1$ in its own free-variable position and $0$s in all others — so they are independent of each other. Together they span $N(A)$. There are $n - r$ free variables:

$$\dim N(A) = n - r$$
The Fundamental Balance

For any $m \times n$ matrix with rank $r$:

  • $\dim C(A) = r$   — the pivot columns
  • $\dim N(A) = n - r$   — the free columns
$$r + (n - r) = n$$

Every column is either a pivot column or a free column. No column is counted in both. The dimensions of the two subspaces partition the full input dimension $n$ exactly.

Worked example

Let

$$A = \begin{bmatrix} 1 & 2 & 3 & 1 \\ 1 & 1 & 2 & 1 \\ 1 & 2 & 3 & 1 \end{bmatrix}$$

Elimination identifies columns 1 and 2 as pivot columns, so $r = 2$. Columns 3 and 4 are free.

  • Basis for $C(A)$: columns 1 and 2 of $A$.   $\dim C(A) = 2$.
  • Basis for $N(A)$: the two special solutions below.   $\dim N(A) = 4 - 2 = 2$.
$$s_1 = \begin{bmatrix}-1\\-1\\1\\0\end{bmatrix}, \qquad s_2 = \begin{bmatrix}-1\\0\\0\\1\end{bmatrix}$$

Check: $2 + 2 = 4 = n$. The balance holds.

6. What the Null Space Actually Means

The null space has a clean algebraic definition — all $x$ with $Ax = 0$ — but it is worth building a geometric picture of what it actually represents. Two pictures are useful: one for $N(A)$ and one for $N(A^T)$.

The walking picture of $N(A)$

Think of each column of $A$ as a step direction in $\mathbb{R}^m$. There are $n$ such directions, one per column. The vector $x = (x_1, x_2, \dots, x_n)$ tells you how many steps to take in each direction — positive or negative. The product $Ax$ is then your final position after all those steps:

$$Ax = x_1 \,\text{col}_1 + x_2 \,\text{col}_2 + \dots + x_n \,\text{col}_n$$

The null space is the set of all step combinations that bring you exactly back to the origin. You walk along the column directions, and no matter which free choices you make, you end up back where you started.

The null space as round trips

$N(A)$ is the set of all step counts $x$ such that the walk $x_1 \,\text{col}_1 + \dots + x_n \,\text{col}_n$ returns to the origin. Every vector in the null space describes a round trip through the column directions.

If the columns are independent, the only round trip is to take no steps at all — $x = 0$. There is no way to wander and come back.

If the columns are dependent, some columns are redundant directions — you can use them to cancel each other out and return to start. Each special solution is one such round trip: it pins one free variable to 1 and records the exact combination of steps that cancels to zero.

This picture reframes dependence naturally. Dependence does not mean "some columns are bad." It means there exists a non-trivial round trip — some combination of steps cancels itself out. The free variables are exactly the choices that let you wander and return, and each special solution is one complete round trip.

The missing directions: $N(A^T)$

The column space $C(A)$ is a subspace of $\mathbb{R}^m$. If $r < m$, it does not fill all of $\mathbb{R}^m$ — some directions in the output space are completely unreachable, no matter how you choose $x$. These are directions perpendicular to every column of $A$.

A direction $y$ in $\mathbb{R}^m$ is unreachable if $y$ is orthogonal to every vector in $C(A)$. That means $y^T(Ax) = 0$ for every $x$, which gives $(A^T y)^T x = 0$ for every $x$, which forces $A^T y = 0$. So the missing directions — the ones you can never reach by combining the columns of $A$ — are exactly the null space of $A^T$.

The left null space as missing directions

The left null space $N(A^T)$ consists of all directions in $\mathbb{R}^m$ that are orthogonal to the entire column space $C(A)$. These are the output directions that $A$ cannot reach — the "blind spots" of the transformation.

$C(A)$ and $N(A^T)$ are orthogonal complements in $\mathbb{R}^m$: every vector in $\mathbb{R}^m$ can be uniquely split into a part in $C(A)$ and a part in $N(A^T)$, and these two parts are perpendicular. Their dimensions sum to $m$:

$$\dim C(A) + \dim N(A^T) = r + (m - r) = m$$

The same relationship holds in the input space $\mathbb{R}^n$: the row space $C(A^T)$ and the null space $N(A)$ are orthogonal complements, and their dimensions sum to $n$. This gives the full picture of four subspaces — two in $\mathbb{R}^m$, two in $\mathbb{R}^n$, paired as perpendicular complements.

Going Further: Bases Beyond Column Vectors

Independence, spanning, and dimension are not properties of $\mathbb{R}^n$ specifically. They apply to any vector space — including spaces whose "vectors" are matrices or functions. The definitions are identical; only the objects change.

Matrix spaces

The space $M$ of all $2 \times 2$ matrices has dimension 4. A natural basis uses the four matrices with a single $1$ and zeros elsewhere:

$$A_1 = \begin{bmatrix}1&0\\0&0\end{bmatrix}, \quad A_2 = \begin{bmatrix}0&1\\0&0\end{bmatrix}, \quad A_3 = \begin{bmatrix}0&0\\1&0\end{bmatrix}, \quad A_4 = \begin{bmatrix}0&0\\0&1\end{bmatrix}$$

Any $2 \times 2$ matrix is a unique combination $c_1 A_1 + c_2 A_2 + c_3 A_3 + c_4 A_4$. The full $n \times n$ matrix space has dimension $n^2$. Subspaces within it:

  • Upper triangular matrices: dimension $\tfrac{1}{2}n^2 + \tfrac{1}{2}n$.
  • Diagonal matrices: dimension $n$.
  • Symmetric matrices: dimension $\tfrac{1}{2}n^2 + \tfrac{1}{2}n$.

Function spaces

Solutions to second-order differential equations also form vector spaces of dimension 2 — because a second-order equation has two free constants. Three examples:

  • $y'' = 0$  →  $y = cx + d$.   Basis: $\{x,\, 1\}$.
  • $y'' = -y$  →  $y = c\sin x + d\cos x$.   Basis: $\{\sin x,\, \cos x\}$.
  • $y'' = y$  →  $y = ce^x + de^{-x}$.   Basis: $\{e^x,\, e^{-x}\}$.

The basis changes, but the dimension is always 2. The same theorem is at work: all bases for a given space contain the same number of vectors. That number is the dimension, and it is a property of the space itself — not of any particular basis you happen to choose.
